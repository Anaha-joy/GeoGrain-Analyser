GeoGrain Analyser
GeoGrain Analyser is a Python-based software tool for automated grain and sediment analysis from remotely sensed imagery. It was developed to support geomorphic interpretation through image-based grain detection, boulder extraction, and grain-size measurement.
Overview
The software is intended for researchers and practitioners working with sediment surfaces, geomorphic mapping, and remotely sensed image datasets. It helps convert imagery into measurable grain information that can be used for analysis, reporting, and interpretation.
Core Capabilities
Automated grain detection from imagery
Grain-size measurement and summary statistics
Boulder and rock extraction support
Machine-learning-assisted rock classification
Generation of overlays and structured output files
Support for geomorphic and sediment analysis workflows
Typical Outputs
Depending on the workflow, GeoGrain Analyser can generate:
Grain boundary overlays
Grain-level measurement tables
Summary grain-size statistics
CSV exports
GeoJSON exports for mapping workflows
Technical Direction
GeoGrain Analyser is built in Python and uses a combination of image processing and machine learning methods to improve sediment feature extraction from imagery. The tool is designed to support practical, repeatable analysis for research and applied geomorphology.
Use Cases
Sediment surface characterization
Grain-size assessment from image data
Boulder detection and mapping
Remote sensing support for geomorphic studies
Research workflows using UAV or other image-based datasets
Project Goal
The goal of GeoGrain Analyser is to make sediment and geomorphic image analysis more efficient, reproducible, and accessible through software-driven workflows.
Status
This project is under active development and continues to evolve with improvements to detection quality, classification, and analysis outputs.
